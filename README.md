# Tree Shaker
A Python project for tree shaking and optimizing JavaScript bundles.

## Usage
1. Load the modules and bundle data using the `load_modules` and `load_bundle` functions.
2. Create a `TreeShaker` instance with the loaded modules.
3. Call the `shake` method to remove unused modules.
4. Call the `optimize` method to optimize the bundle.

## Testing
Run the tests using `pytest`.
