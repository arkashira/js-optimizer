# 🚀 js-optimizer
**Power developers with optimized JavaScript code.** A Python project for optimizing JavaScript code by removing unused exports and pre-compiling critical paths.

## Why js-optimizer?
* **Faster Execution**: Optimize JavaScript code for faster execution and improved performance.
* **Reduced Size**: Remove unused exports to reduce the overall size of the codebase.
* **Improved Maintainability**: Pre-compile critical paths to simplify maintenance and updates.
* **Easy Integration**: Designed as a command-line tool or library for easy integration into existing workflows.
* **Python-Powered**: Leverages the power and flexibility of Python for optimization tasks.
* **JavaScript Focus**: Specifically designed for optimizing JavaScript code, ensuring the best possible results.
* **Open-Source**: Free and open-source, allowing for community contributions and customization.

## Feature Overview
| Feature | Description |
| --- | --- |
| Unused Export Removal | Automatically removes unused exports from JavaScript code |
| Critical Path Pre-Compilation | Pre-compiles critical paths for improved performance and maintainability |
| Command-Line Interface | Easy-to-use command-line interface for integration into existing workflows |
| Library Integration | Can be used as a library for programmatic optimization of JavaScript code |

## Getting Started
To get started with js-optimizer, follow these steps:
```bash
# Clone the repository
git clone https://github.com/axentx/js-optimizer.git

# Install dependencies
pip install -r requirements.txt

# Run the optimizer
python src/js_optimizer.py --help
```

## Deploy
To deploy js-optimizer, use the following command:
```bash
# Build the package
python -m build

# Install the package
pip install dist/js_optimizer-1.0.0.tar.gz
```
