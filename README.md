# TurboJS Optimizer

This project integrates TurboJS with Webpack to optimize JavaScript files.

## Usage

1. Create a `TurboJSConfig` object with the entry point and output file.
2. Call the `integrate_turbojs_with_webpack` function with the `TurboJSConfig` object and a list of JavaScript files.
3. The function returns a Webpack configuration string that uses TurboJS as the loader.
